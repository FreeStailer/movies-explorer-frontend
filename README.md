1. Express;
2. Mongo DB;
3. Обработка ошибок celebrate;
4. Логирование ошибок и запросов express-winston;
5. express-rate-limit;
6. dotenv;
7. Хеширование паролей bcryptjs;
8. Проверка eslint;
9. Helmet;
10. validator;
11. nodemon;
